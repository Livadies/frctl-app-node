# Проверка FRCTL Hub инвестором

## Рабочая ссылка

https://livadies-frctl-hub.static.hf.space/

## Основной тест

1. Дождитесь статуса `LIVE CATALOG` и счётчика 48 карточек.
2. Выберите «ИИ и модели» и найдите `text-generation`.
3. Выберите `RustDesk`.
4. Введите `10.0.0.12:21118`.
5. Выберите «Подключить» и нажмите «Сформировать dry-run».

Ожидается план из пяти защитных шагов без реального сетевого подключения.

## Проверка безопасности

Введите `root:password@server.example`. Приложение должно отклонить credentials.

## Проверка автономности

После загрузки страницы отключите интернет и повторите основной тест. Планировщик продолжит работать, поскольку он выполняется локально в браузере.

## Что проверяется

- единый живой marketplace GitHub + Hugging Face с категориями и кэшем;
- policy‑планирование;
- local-first персонализация;
- отсутствие серверного сбора целей и истории;
- безопасная граница будущего локального Node.

FRCTL Hub является MVP и пока не выполняет удалённые команды и не имеет сертификации ФСТЭК.
