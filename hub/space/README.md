# FRCTL Hub

Public, non-privileged demonstration of the FRCTL control plane. It plans remote-access workflows, shows a connector marketplace, keeps adaptive suggestions in the browser, and accepts only consented, redacted telemetry.

This Space does not execute remote commands, store credentials, provide an unattended agent, or claim FSTEC certification.
